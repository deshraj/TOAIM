Instructions

1. The question will remain the same across all images in the HIT.

2. Please note that Vicki can say only one of a 1000 things which are listed in the dropdown. So there are only 1000 "valid answers".

3. As you type in the textbox, the dropdown will be updated to show the list of valid answers based on the text that you entered. Please select a valid answer. The interface will let you progress to the next task only if the answer is a valid answer (something that Vicki can say).

4. There will be two phases in this HIT. In the first phase, after each question, we will show you what Vicki's actual answer was. We hope that this will give you a better sense for Vicki’s behavior and help you make more accurate guesses.

5. The second phase will be a test phase, in which the task will be the same but we will no longer show you Vicki's actual answer. We will pay you a bonus if you do a good job with making accurate guesses!

6. Please note that at times, a question may not be relevant to the image e.g., "What color is the cat?" for an image with no cat in it. In such cases, Vicki still had to answer -- making its best guess among one of the 1000 valid answers that made the most sense. For instance, it might decide to name the color of the thing that looks most like a cat or name the dominant color of the image, etc. In these cases, you'll have to guess what Vicki's guess most likely was.
